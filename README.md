## 功能介绍
### 本小程序意愿是在实现任务发布及其审核机制主题下，完成了这个小程序的开发，还有许多未开发得功能，正在逐步完善，意愿成为一款‘睿智’小程序。

## 功能点

* 计划发布
   * 打卡任务
   * 运动连续性
   * 运动累积性
* 打卡任务审核
   * 审核通知
* 运动型打卡
   * 根据每日微信运动打卡
* 计划管理
* 任务管理
* 卡包功能（实现中...）
    * 初衷是为了记住银行卡账号
* 计划搜索 (实现中...)
